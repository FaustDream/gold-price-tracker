    - [x] 锁定位置

    - [x] 置顶显示
        - [x] 提供“始终置顶”开关：启用后窗口保持在所有软件之上显示（不抢焦点）。

    - [x] 穿透点击（置顶后可选）
        - [x] 提供“鼠标穿透”开关：启用后窗口可见但不拦截鼠标事件，鼠标可直接点击下方应用。
